# Advanced BMI Calculator Web App

A complete BMI Calculator web application developed using:

- Python
- Flask
- SQLite
- HTML
- CSS
- Matplotlib

## Features

- BMI Calculation
- BMI Categories
- Input Validation
- Database Storage
- Historical Data Viewing
- BMI Graph Analysis
- User-Friendly Interface

## Install Dependencies

pip install -r requirements.txt

## Run Project

python app.py

## Open Browser

http://127.0.0.1:5000
